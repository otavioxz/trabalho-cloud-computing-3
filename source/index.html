Portal de Notícias Cloud

Informação em tempo real, hospedada em um ambiente Linux automatizado.

Início Sobre o projeto
Tecnologia

Como o DevOps simplifica a publicação de notícias

Este portal roda em um container Ubuntu Server com Apache, provisionado por scripts Shell. Backups, deploys e monitoramento são automatizados, simulando a rotina de uma equipe de operações de cloud.

Cloud

Infraestrutura como código

O Dockerfile e o docker-compose descrevem todo o ambiente do portal.

Operações

Backups diários

Pastas de artigos, autores e categorias são empacotadas em tar.gz datados.

Editorial

Grupo noticias_ops

Usuários jornalista e editor_chefe têm permissões controladas via chmod/chown.

Monitoramento

Alertas em tempo real

CPU, RAM, disco e status do Apache verificados a cada execução.
